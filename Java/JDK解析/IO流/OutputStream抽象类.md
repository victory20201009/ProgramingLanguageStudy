1. `java.io.OutputStream`类是一个抽象类，是所有字节输出流的基类，这样的对象可以接受输出的字节，并把它们发送到某个接收器。

   > /* Applications that need to define a subclass of

   > * that writes one byte of output. 
   > * 定义一个OutputStream的子类，必须至少实现写一个方法可以写一个字节
   > */
   > ```
